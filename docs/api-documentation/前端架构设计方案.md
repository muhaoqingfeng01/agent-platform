## 一、项目概述

### 1.1 背景

Agent Platform 是一个企业级 AI 代理平台，提供以下核心能力：

- **对话管理**: 流式聊天、会话历史、消息反馈
- **知识库引擎**: RAG检索、文档管理、混合搜索
- **任务规划**: DAG任务编排、工具调用、执行监控
- **权限体系**: 多租户、RBAC角色权限、审批工作流
- **安全治理**: 敏感词过滤、安全事件审计
- **效果评估**: BadCase闭环、优化工单、数据集评测
- **可观测性**: 全链路追踪、日志分析、性能监控

### 1.2 设计目标

| 目标 | 说明 |
|------|------|
| **前后端分离** | 纯静态资源部署，通过 API 与后端交互 |
| **配置化驱动** | 请求域名、超时时间、功能开关全部可配置 |
| **模块化架构** | 按业务域拆分模块，支持按需加载和扩展 |
| **类型安全** | TypeScript 全量覆盖，API 接口自动生成类型定义 |
| **响应式设计** | 适配桌面端和移动端（优先桌面端） |
| **实时通信** | WebSocket + SSE 双通道支持流式对话和实时推送 |

## 二、技术选型

### 2.1 核心技术栈

| 类别 | 技术 | 版本 | 用途 |
|------|------|------|------|
| **框架** | React | 18.x | UI 组件库 |
| **语言** | TypeScript | 5.x | 类型系统 |
| **构建工具** | Vite | 5.x | 快速开发和打包 |
| **UI 组件库** | Ant Design | 5.x | 企业级 UI 组件 |
| **状态管理** | Zustand | 4.x | 轻量级全局状态 |
| **路由** | React Router | 6.x | 客户端路由 |
| **HTTP 客户端** | Axios | 1.x | REST API 调用 |
| **SSE 客户端** | EventSource | 原生 | 流式对话接收 |
| **WebSocket** | native WebSocket | 原生 | 实时消息推送 |
| **表单管理** | React Hook Form | 7.x | 高性能表单处理 |
| **数据获取** | TanStack Query | 5.x | 服务端状态缓存 |
| **虚拟滚动** | react-window | 3.x | 长列表性能优化 |
| **代码规范** | ESLint + Prettier | - | 代码质量保障 |
| **图标库** | @ant-design/icons | 5.x | 统一图标风格 |

### 2.2 为什么不选择 Redux/MobX？

- **Zustand 优势**: 
  - 更简洁的 API（基于 Hooks）
  - 无样板代码，直接操作 state
  - 天然支持 TypeScript
  - 体积更小（~1KB）
  - 适合中小型项目

```typescript
// src/config/app.config.ts
interface AppConfig {
  api: {
    baseURL: string;
    timeout: number;
    retryCount: number;
    retryDelay: number;
  };
  websocket: {
    url: string;
    heartbeatInterval: number;
    reconnectAttempts: number;
    reconnectDelay: number[];
  };
  features: Record<string, boolean>;
  chat: {
    maxMessageLength: number;
    pageSize: number;
    streamTimeout: number;
  };
  upload: {
    maxFileSize: number;
    allowedTypes: string[];
  };
}

let appConfig: AppConfig | null = null;

export async function loadAppConfig(): Promise<AppConfig> {
  if (appConfig) return appConfig;
  
  const response = await fetch('/config.json');
  appConfig = await response.json();
  return appConfig;
}

export function getAppConfig(): AppConfig {
  if (!appConfig) {
    throw new Error('App config not loaded. Call loadAppConfig() first.');
  }
  return appConfig;
}
```

**优势**:
- 无需重新构建即可修改配置
- 支持不同客户定制化配置
- 运维友好

### 4.2 Axios 实例配置

```typescript
// src/services/apiClient.ts
import axios, { AxiosInstance, AxiosError } from 'axios';
import { getAppConfig } from '@/config/app.config';
import { useAuthStore } from '@/stores/useAuthStore';
import { message } from 'antd';

class ApiClient {
  private instance: AxiosInstance;

  constructor() {
    const config = getAppConfig();
    
    this.instance = axios.create({
      baseURL: config.api.baseURL,
      timeout: config.api.timeout,
      headers: {
        'Content-Type': 'application/json',
      },
    });

    // 请求拦截器
    this.instance.interceptors.request.use(
      (config) => {
        const token = useAuthStore.getState().token;
        if (token) {
          config.headers.Authorization = `Bearer ${token}`;
        }
        return config;
      },
      (error) => Promise.reject(error)
    );

    // 响应拦截器
    this.instance.interceptors.response.use(
      (response) => response.data,
      (error: AxiosError) => {
        if (error.response?.status === 401) {
          useAuthStore.getState().logout();
          window.location.href = '/login';
          message.error('登录已过期，请重新登录');
        } else if (error.response?.status === 403) {
          message.error('权限不足');
        } else if (error.response?.status >= 500) {
          message.error('服务器错误，请稍后重试');
        }
        return Promise.reject(error);
      }
    );
  }

  getInstance(): AxiosInstance {
    return this.instance;
  }
}

export const apiClient = new ApiClient().getInstance();
```

```typescript
import { apiClient } from './apiClient';
import type { 
  Conversation, 
  CreateConversationRequest,
  UpdateConversationTitleRequest,
  PageResponse 
} from '@/types/conversation';

export const conversationService = {
  // 创建会话
  create: (data: CreateConversationRequest) => 
    apiClient.post<Conversation>('/api/v1/conversations', data),

  // 会话列表
  list: (page: number = 0, size: number = 20) =>
    apiClient.get<PageResponse<Conversation>>('/api/v1/conversations', {
      params: { page, size }
    }),

  // 会话详情
  getById: (id: string) =>
    apiClient.get<Conversation>(`/api/v1/conversations/${id}`),

  // 更新标题
  updateTitle: (id: string, title: string) =>
    apiClient.patch(`/api/v1/conversations/${id}/title`, { title }),

  // 状态流转
  transitionStatus: (id: string, targetStatus: string) =>
    apiClient.patch(`/api/v1/conversations/${id}/status`, { targetStatus }),

  // 删除会话
  delete: (id: string) =>
    apiClient.delete(`/api/v1/conversations/${id}`),
};
```

### 5.3 SSE 流式对话实现

```typescript
// src/hooks/useSSE.ts
import { useState, useCallback, useRef } from 'react';
import { getAppConfig } from '@/config/app.config';
import { useAuthStore } from '@/stores/useAuthStore';

interface SSEOptions {
  onMessage?: (data: any) => void;
  onError?: (error: Event) => void;
  onOpen?: () => void;
}

export function useSSE(options: SSEOptions = {}) {
  const [isLoading, setIsLoading] = useState(false);
  const eventSourceRef = useRef<EventSource | null>(null);
  const config = getAppConfig();
  const token = useAuthStore(state => state.token);

  const connect = useCallback((url: string, body: any) => {
    setIsLoading(true);
    
    // SSE 不支持 POST，需要通过特殊方式传递参数
    // 方案1: URL参数（注意长度限制）
    // 方案2: 先POST获取eventId，再GET订阅
    // 本项目采用方案2
    
    const eventSource = new EventSource(
      `${config.api.baseURL}${url}?token=${token}`,
      { withCredentials: true }
    );

    eventSource.onopen = () => {
      options.onOpen?.();
    };

    eventSource.onmessage = (event) => {
      try {
        const data = JSON.parse(event.data);
        options.onMessage?.(data);
        
        // 流式结束标记
        if (data.type === 'done') {
          eventSource.close();
          setIsLoading(false);
        }
      } catch (error) {
        console.error('SSE parse error:', error);
      }
    };

    eventSource.onerror = (error) => {
      options.onError?.(error);
      eventSource.close();
      setIsLoading(false);
    };

    eventSourceRef.current = eventSource;
  }, [token, options]);

  const disconnect = useCallback(() => {
    if (eventSourceRef.current) {
      eventSourceRef.current.close();
      eventSourceRef.current = null;
      setIsLoading(false);
    }
  }, []);

  return { isLoading, connect, disconnect };
}
```

### 5.4 WebSocket 实时推送实现

```typescript
// src/hooks/useWebSocket.ts
import { useState, useEffect, useCallback, useRef } from 'react';
import { getAppConfig } from '@/config/app.config';
import { useAuthStore } from '@/stores/useAuthStore';

interface WebSocketMessage {
  type: string;
  payload: any;
  timestamp: number;
}

interface UseWebSocketOptions {
  onMessage?: (message: WebSocketMessage) => void;
  onOpen?: () => void;
  onClose?: () => void;
  onError?: (error: Event) => void;
}

export function useWebSocket(options: UseWebSocketOptions = {}) {
  const [isConnected, setIsConnected] = useState(false);
  const wsRef = useRef<WebSocket | null>(null);
  const reconnectTimerRef = useRef<NodeJS.Timeout | null>(null);
  const config = getAppConfig();
  const token = useAuthStore(state => state.token);
  const reconnectAttempts = useRef(0);

  const connect = useCallback(() => {
    if (!token) return;

    const wsUrl = `${config.websocket.url}?token=${token}`;
    const ws = new WebSocket(wsUrl);

    ws.onopen = () => {
      console.log('[WebSocket] Connected');
      setIsConnected(true);
      reconnectAttempts.current = 0;
      options.onOpen?.();

      // 启动心跳
      startHeartbeat(ws);
    };

    ws.onmessage = (event) => {
      try {
        const message: WebSocketMessage = JSON.parse(event.data);
        
        // 处理心跳响应
        if (message.type === 'pong') return;

        options.onMessage?.(message);
      } catch (error) {
        console.error('[WebSocket] Parse error:', error);
      }
    };

    ws.onclose = () => {
      console.log('[WebSocket] Closed');
      setIsConnected(false);
      options.onClose?.();
      
      // 自动重连
      scheduleReconnect();
    };

    ws.onerror = (error) => {
      console.error('[WebSocket] Error:', error);
      options.onError?.(error);
    };

    wsRef.current = ws;
  }, [token, options]);

  const disconnect = useCallback(() => {
    if (wsRef.current) {
      wsRef.current.close();
      wsRef.current = null;
    }
    if (reconnectTimerRef.current) {
      clearTimeout(reconnectTimerRef.current);
      reconnectTimerRef.current = null;
    }
  }, []);

  const sendMessage = useCallback((data: any) => {
    if (wsRef.current?.readyState === WebSocket.OPEN) {
      wsRef.current.send(JSON.stringify(data));
    }
  }, []);

  // 心跳检测
  const startHeartbeat = (ws: WebSocket) => {
    const interval = setInterval(() => {
      if (ws.readyState === WebSocket.OPEN) {
        ws.send('ping');
      } else {
        clearInterval(interval);
      }
    }, config.websocket.heartbeatInterval);
  };

  // 指数退避重连
  const scheduleReconnect = () => {
    if (reconnectAttempts.current >= config.websocket.reconnectAttempts) {
      console.warn('[WebSocket] Max reconnect attempts reached');
      return;
    }

    const delay = config.websocket.reconnectDelay[reconnectAttempts.current] || 16000;
    reconnectTimerRef.current = setTimeout(() => {
      reconnectAttempts.current++;
      connect();
    }, delay);
  };

  useEffect(() => {
    connect();
    return () => disconnect();
  }, [connect, disconnect]);

  return { isConnected, sendMessage, disconnect };
}
```

```typescript
interface ChatState {

  currentMessage: string;
```

```tsx
      );
```

```typescript
// src/utils/logger.ts
import { getAppConfig } from '@/config/app.config';

enum LogLevel {
  DEBUG = 'DEBUG',
  INFO = 'INFO',
  WARN = 'WARN',
  ERROR = 'ERROR',
}

class Logger {
  private logToServer(level: LogLevel, message: string, meta?: any) {
    // 生产环境上报到服务器
    if (getAppConfig().api.env === 'production') {
      navigator.sendBeacon('/api/v1/logs', JSON.stringify({
        level,
        message,
        meta,
        timestamp: Date.now(),
        url: window.location.href,
        userAgent: navigator.userAgent,
      }));
    }
  }

  error(message: string, meta?: any) {
    console.error(`[ERROR] ${message}`, meta);
    this.logToServer(LogLevel.ERROR, message, meta);
  }

  warn(message: string, meta?: any) {
    console.warn(`[WARN] ${message}`, meta);
    this.logToServer(LogLevel.WARN, message, meta);
  }

  info(message: string, meta?: any) {
    console.info(`[INFO] ${message}`, meta);
  }

  debug(message: string, meta?: any) {
    if (getAppConfig().api.env === 'development') {
      console.debug(`[DEBUG] ${message}`, meta);
    }
  }
}

export const logger = new Logger();
```
