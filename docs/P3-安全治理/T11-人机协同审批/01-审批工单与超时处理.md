# 人机协同审批

## 所属阶段
**P3 安全治理 → T11 人机协同审批**

## 使用技术
- 自研轻量审批工作流
- WebSocket 推送审批卡片
- Spring `@Scheduled` 超时扫描

## 涉及数据库表
- `t_approval_workflow`

## API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/approvals` | 审批列表（我的待审批/我已审批） |
| GET | `/api/v1/approvals/{id}` | 审批详情 |
| POST | `/api/v1/approvals/{id}/approve` | 同意 |
| POST | `/api/v1/approvals/{id}/reject` | 拒绝（需填写原因） |
| GET | `/api/v1/approvals/stats` | 审批统计 |

## 实现方案

### 1. 审批流程

```
高风险工具调用
    │
    ▼
┌────────────────┐
│ 生成审批工单     │ ← approval_id, 操作内容, 请求人, timeout=5min
│ status=PENDING │
└──────┬─────────┘
       │
       ▼
┌────────────────┐
│ WebSocket 推送  │ → 推给所有拥有 APPROVER 角色的在线用户
└──────┬─────────┘
       │
  ┌────┴────┐
  ▼         ▼
同意      拒绝/超时
  │         │
  ▼         ▼
继续执行   终止任务
  │         │
  ▼         ▼
通知请求人  通知请求人
```

### 2. ApprovalWorkflowService

```java
@Service
@Transactional
public class ApprovalWorkflowService {

    private static final int TIMEOUT_MINUTES = 5;

    public ApprovalWorkflow createApproval(String toolId, String executionId,
                                            String conversationId, Map<String, Object> params) {
        ToolRegistry tool = toolRepo.findByToolId(toolId);
        String approverId = assignApprover();  // 轮询分配或推送给所有审批人

        ApprovalWorkflow approval = ApprovalWorkflow.builder()
                .approvalId(IdGenerator.generate("appr"))
                .tenantId(TenantContext.getCurrentTenantId())
                .toolId(toolId)
                .conversationId(conversationId)
                .executionId(executionId)
                .requesterId(TenantContext.getCurrentUserId())
                .approverId(approverId)
                .title("工具调用审批: " + tool.getName())
                .operationDetail(JsonUtil.toJson(Map.of(
                    "toolName", tool.getName(),
                    "params", params,
                    "riskLevel", "HIGH"
                )))
                .status("PENDING")
                .timeoutAt(LocalDateTime.now().plusMinutes(TIMEOUT_MINUTES))
                .build();
        approvalRepository.save(approval);

        // 推送审批卡片
        pushApprovalCard(approval);

        return approval;
    }

    public void approve(String approvalId, String comment) {
        ApprovalWorkflow approval = approvalRepository.findByApprovalId(approvalId);
        validatePending(approval);

        approval.setStatus("APPROVED");
        approval.setApproveComment(comment);
        approval.setApprovedAt(LocalDateTime.now());
        approvalRepository.updateById(approval);

        // 回调执行引擎继续
        dagExecutor.resumeExecution(approval.getExecutionId());
        pushResult(approval, "APPROVED");
    }

    public void reject(String approvalId, String reason) {
        ApprovalWorkflow approval = approvalRepository.findByApprovalId(approvalId);
        validatePending(approval);

        approval.setStatus("REJECTED");
        approval.setApproveComment(reason);
        approval.setApprovedAt(LocalDateTime.now());
        approvalRepository.updateById(approval);

        // 终止任务
        dagExecutor.cancelExecution(approval.getExecutionId(), "审批拒绝: " + reason);
        pushResult(approval, "REJECTED");
    }
}
```

### 3. 超时扫描

```java
@Component
public class ApprovalTimeoutJob {

    @Scheduled(fixedDelay = 30_000)  // 每30秒扫描一次
    public void scanTimeout() {
        List<ApprovalWorkflow> timeouts = approvalRepository
                .findByStatusAndTimeoutBefore("PENDING", LocalDateTime.now());

        for (ApprovalWorkflow approval : timeouts) {
            approval.setStatus("TIMEOUT");
            approval.setApproveComment("审批超时，系统自动拒绝");
            approvalRepository.updateById(approval);

            // 终止任务
            dagExecutor.cancelExecution(approval.getExecutionId(), "审批超时");
            pushResult(approval, "TIMEOUT");
            log.warn("审批超时自动拒绝: {}", approval.getApprovalId());
        }
    }
}
```
