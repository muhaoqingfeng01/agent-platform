# 提示词运行时渲染

## 所属阶段
**P1 核心能力 → T4 提示词管理与版本控制**

## 使用技术
- Spring AI `PromptTemplate`
- 自定义变量解析器
- Redis 缓存热点模板

## 实现方案

### 1. 运行时渲染流程

```
编排引擎请求: getPrompt("agent_system", context)
    │
    ▼
┌──────────────────┐
│ 1. 查 Redis 缓存  │ ← key: prompt:{tenantId}:{name}:{version}
└──────┬───────────┘
       │ 未命中
       ▼
┌──────────────────┐
│ 2. 查 MySQL      │ ← t_prompt_template WHERE name=? AND status='PUBLISHED'
└──────┬───────────┘
       │
       ▼
┌──────────────────┐
│ 3. 变量替换       │ ← 遍历 context，填充 {{variable}}
└──────┬───────────┘
       │
       ▼
┌──────────────────┐
│ 4. 返回最终 Prompt │
└──────────────────┘
```

### 2. 核心代码

```java
@Service
public class PromptRenderService {

    private final PromptRepository promptRepository;
    private final RedisTemplate<String, String> redisTemplate;

    /**
     * 根据名称获取已发布模板并渲染
     */
    public String render(String name, int version, Map<String, Object> context) {
        // 1. 查缓存
        String cacheKey = buildCacheKey(name, version);
        String cachedTemplate = redisTemplate.opsForValue().get(cacheKey);

        String templateText;
        if (cachedTemplate != null) {
            templateText = cachedTemplate;
        } else {
            // 2. 查数据库
            PromptTemplate template = promptRepository.findByNameAndVersion(name, version);
            if (template == null || !"PUBLISHED".equals(template.getStatus())) {
                throw new BusinessException(404, "提示词模板不存在或未发布: " + name);
            }
            templateText = template.getTemplateText();
            // 3. 写入缓存（10分钟）
            redisTemplate.opsForValue().set(cacheKey, templateText, 10, TimeUnit.MINUTES);
        }

        // 4. 变量替换
        return applyVariables(templateText, context);
    }

    private String applyVariables(String template, Map<String, Object> context) {
        String result = template;
        for (Map.Entry<String, Object> entry : context.entrySet()) {
            String placeholder = "{{" + entry.getKey() + "}}";
            String value = formatValue(entry.getValue());
            result = result.replace(placeholder, value);
        }
        return result;
    }

    private String formatValue(Object value) {
        if (value == null) return "";
        if (value instanceof List<?> list) {
            return list.stream().map(Object::toString).collect(Collectors.joining("\n"));
        }
        if (value instanceof Map<?, ?> map) {
            return JsonUtil.toJson(map);
        }
        return value.toString();
    }

    private String buildCacheKey(String name, int version) {
        return String.format("prompt:%s:%s:v%d",
                TenantContext.getCurrentTenantId(), name, version);
    }
}
```

### 3. 特殊变量（系统注入）

以下变量由系统自动注入，无需在模板中声明：

| 变量名 | 来源 | 说明 |
|--------|------|------|
| `{{user_name}}` | Sa-Token | 当前用户名 |
| `{{tenant_name}}` | t_tenant 表 | 租户名称 |
| `{{current_time}}` | 系统时钟 | 当前时间格式化 |
| `{{conversation_history}}` | Redis | 最近对话历史 |
| `{{knowledge_context}}` | RAG 检索 | 知识库检索结果 Top K |
| `{{available_tools}}` | MCP Client | 可用工具清单 |
