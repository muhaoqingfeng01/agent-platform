# 意图识别引擎

## 所属阶段
**P1 核心能力 → T3 意图识别与对话管理**

## 使用技术
- Spring AI（LLM 意图分类）
- 正则/关键词匹配引擎
- Redis（意图缓存）

## 涉及数据库表
- `t_intent`

## 实现方案

### 1. 混合识别策略

```
用户输入
    │
    ▼
┌─────────────┐
│ 1. 规则匹配  │ ← 正则/关键词（~1ms）
└──────┬──────┘
       │ 命中? ──→ 返回意图
       │ 未命中
       ▼
┌─────────────┐
│ 2. Redis缓存 │ ← 相似问法缓存（~5ms）
└──────┬──────┘
       │ 命中? ──→ 返回意图
       │ 未命中
       ▼
┌─────────────┐
│ 3. LLM 分类  │ ← 兜底，返回意图枚举（~500ms）
└─────────────┘
```

### 2. IntentRecognitionService

```java
@Service
@Slf4j
public class IntentRecognitionService {

    private final IntentRepository intentRepository;
    private final ChatClient chatClient;
    private final RedisTemplate<String, String> redisTemplate;

    public IntentResult recognize(String userInput) {
        String tenantId = TenantContext.getCurrentTenantId();

        // Layer 1: 规则匹配
        IntentResult ruleResult = matchByRules(tenantId, userInput);
        if (ruleResult != null && ruleResult.getConfidence() >= 0.9) {
            return ruleResult;
        }

        // Layer 2: 缓存查找
        String cacheKey = "t_intent:cache:" + tenantId + ":" + hash(userInput);
        String cached = redisTemplate.opsForValue().get(cacheKey);
        if (cached != null) {
            return JsonUtil.fromJson(cached, IntentResult.class);
        }

        // Layer 3: LLM 兜底
        IntentResult llmResult = classifyByLLM(tenantId, userInput);
        redisTemplate.opsForValue().set(cacheKey, JsonUtil.toJson(llmResult), 30, TimeUnit.MINUTES);
        return llmResult;
    }

    private IntentResult matchByRules(String tenantId, String input) {
        List<Intent> intents = intentRepository.findActiveByTenant(tenantId);
        for (Intent t_intent : intents) {
            List<String> patterns = t_intent.getPatterns();
            for (String pattern : patterns) {
                if (Pattern.matches(pattern, input)) {
                    return IntentResult.matched(t_intent, 1.0);
                }
            }
            // 关键词匹配（每个关键词匹配+0.1 置信度，最高 0.9）
            double keywordScore = matchKeywords(t_intent.getPatterns(), input);
            if (keywordScore >= 0.7) {
                return IntentResult.matched(t_intent, keywordScore);
            }
        }
        return null;
    }

    private IntentResult classifyByLLM(String tenantId, String input) {
        List<Intent> intents = intentRepository.findActiveByTenant(tenantId);
        String prompt = buildClassificationPrompt(intents, input);
        String response = chatClient.prompt().user(prompt).call().content();
        return parseIntentFromLLM(response);
    }
}
```

### 4. 意图定义 API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/intents` | 创建意图 |
| GET | `/api/v1/intents` | 意图列表 |
| PUT | `/api/v1/intents/{id}` | 编辑意图（规则/示例/LLM提示词） |
| PATCH | `/api/v1/intents/{id}/status` | 启停意图 |
| POST | `/api/v1/intents/{id}/test` | 测试意图识别（输入文本 → 返回匹配结果） |
