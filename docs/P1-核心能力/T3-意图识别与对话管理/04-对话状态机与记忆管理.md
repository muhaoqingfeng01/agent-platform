# 对话状态机与记忆管理

## 所属阶段
**P1 核心能力 → T3 意图识别与对话管理**

## 使用技术
- Spring StateMachine（状态流转）
- Redis（短期记忆）
- MySQL（长期记忆）

## 涉及数据库表
- `t_long_term_memory`

## 实现方案

### 1. 对话状态机

```
                  ┌──────────┐
                  │   INIT   │ ← 会话开始
                  └────┬─────┘
                       │ 用户发送消息
                       ▼
                  ┌──────────┐
                  │AWAIT_PARAM│ ← 等待参数补充
                  └────┬─────┘
                       │ 参数完整
                       ▼
                  ┌──────────┐
                  │EXECUTING │ ← 执行中（调LLM/工具）
                  └────┬─────┘
                       │ 需要确认?
              ┌────────┼────────┐
              ▼        ▼        ▼
         ┌────────┐ ┌──────┐ ┌──────┐
         │CONFIRM │ │ DONE │ │ERROR │
         └────────┘ └──────┘ └──────┘
```

```java
@Configuration
@EnableStateMachine
public class ConversationStateMachineConfig extends StateMachineConfigurerAdapter<String, String> {

    @Override
    public void configure(StateMachineStateConfigurer<String, String> states) {
        states.withStates()
            .initial("INIT")
            .state("AWAIT_PARAM")
            .state("EXECUTING")
            .state("CONFIRMATION")
            .end("DONE")
            .end("ERROR");
    }

    @Override
    public void configure(StateMachineTransitionConfigurer<String, String> transitions) {
        transitions
            .withExternal().source("INIT").target("AWAIT_PARAM").event("MSG_RECEIVED")
            .and()
            .withExternal().source("AWAIT_PARAM").target("EXECUTING").event("PARAMS_COMPLETE")
            .and()
            .withExternal().source("EXECUTING").target("CONFIRMATION").event("NEED_CONFIRM")
            .and()
            .withExternal().source("EXECUTING").target("DONE").event("COMPLETED")
            .and()
            .withExternal().source("CONFIRMATION").target("EXECUTING").event("CONFIRMED")
            .and()
            .withExternal().source("CONFIRMATION").target("DONE").event("CANCELLED");
    }
}
```

### 2. 短期记忆（Redis）

```
Key: session:{tenantId}:{sessionId}
Value: List<Message> (最近 N 轮)
TTL: 30 分钟（每次交互刷新）
```

```java
@Service
public class SessionMemoryService {

    private static final int MAX_ROUNDS = 20;   // 保留最近 20 轮
    private static final int TTL_MINUTES = 30;

    public void appendMessage(String conversationId, Message t_message) {
        String key = "session:" + TenantContext.getCurrentTenantId() + ":" + conversationId;
        redisTemplate.opsForList().rightPush(key, JsonUtil.toJson(t_message));
        redisTemplate.opsForList().trim(key, -MAX_ROUNDS * 2, -1);  // 每轮2条(user+assistant)
        redisTemplate.expire(key, Duration.ofMinutes(TTL_MINUTES));
    }

    public List<Message> getRecentMessages(String conversationId, int rounds) {
        String key = "session:" + TenantContext.getCurrentTenantId() + ":" + conversationId;
        List<String> items = redisTemplate.opsForList().range(key, -rounds * 2, -1);
        return items.stream().map(s -> JsonUtil.fromJson(s, Message.class)).toList();
    }
}
```

### 3. 长期记忆（MySQL）

```java
@Service
public class LongTermMemoryService {

    // 对话结束后异步提取关键信息
    @Async
    public void extractAndSave(String conversationId, String userId) {
        List<Message> history = sessionMemoryService.getRecentMessages(conversationId, 20);
        String summary = summarizeMemories(history);
        List<MemoryFact> facts = parseFactsFromSummary(summary);

        for (MemoryFact fact : facts) {
            LongTermMemory memory = LongTermMemory.builder()
                    .tenantId(TenantContext.getCurrentTenantId())
                    .userId(userId)
                    .memoryType(fact.getType())   // FACT / PREFERENCE / CONTEXT
                    .memoryKey(fact.getKey())
                    .memoryValue(fact.getValue())
                    .confidence(fact.getConfidence())
                    .source(conversationId)
                    .build();
            memoryRepository.upsert(memory);  // key 相同则更新置信度
        }
    }

    // 新会话开始时加载用户记忆
    public List<LongTermMemory> loadUserMemories(String userId) {
        return memoryRepository.findByUserId(userId);
    }
}
```
