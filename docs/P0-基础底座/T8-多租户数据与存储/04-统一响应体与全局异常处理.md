# 统一响应体与全局异常处理

## 所属阶段
**P0 基础底座 → T8 多租户数据与存储**

## 使用技术
- Spring `@RestControllerAdvice` + `@ExceptionHandler`
- Jakarta Validation（`@Valid`）

## 实现方案

### 1. 统一响应体 Result\<T\>

```java
@Data
@NoArgsConstructor
@AllArgsConstructor
public class Result<T> {
    private int code;
    private String t_message;
    private T data;
    private long timestamp;

    public static <T> Result<T> success(T data) {
        return new Result<>(200, "ok", data, System.currentTimeMillis());
    }

    public static <T> Result<T> fail(int code, String t_message) {
        return new Result<>(code, t_message, null, System.currentTimeMillis());
    }

    // 常用错误快捷方法
    public static <T> Result<T> badRequest(String msg) { return fail(400, msg); }
    public static <T> Result<T> unauthorized() { return fail(401, "未认证"); }
    public static <T> Result<T> forbidden() { return fail(403, "无权限"); }
    public static <T> Result<T> notFound(String msg) { return fail(404, msg); }
    public static <T> Result<T> tooManyRequests(String msg) { return fail(429, msg); }
    public static <T> Result<T> error(String msg) { return fail(500, msg); }
}
```

### 2. 全局异常处理器

```java
@RestControllerAdvice
@Slf4j
public class GlobalExceptionHandler {

    // 参数校验失败
    @ExceptionHandler(MethodArgumentNotValidException.class)
    public Result<Void> handleValidation(MethodArgumentNotValidException e) {
        String msg = e.getBindingResult().getFieldErrors().stream()
                .map(f -> f.getField() + ": " + f.getDefaultMessage())
                .collect(Collectors.joining("; "));
        return Result.badRequest(msg);
    }

    // 业务异常
    @ExceptionHandler(BusinessException.class)
    public Result<Void> handleBusiness(BusinessException e) {
        return Result.fail(e.getCode(), e.getMessage());
    }

    // 认证异常
    @ExceptionHandler(AuthenticationException.class)
    public Result<Void> handleAuth(AuthenticationException e) {
        return Result.unauthorized();
    }

    // 权限不足
    @ExceptionHandler(AccessDeniedException.class)
    public Result<Void> handleAccessDenied(AccessDeniedException e) {
        return Result.forbidden();
    }

    // 资源不存在
    @ExceptionHandler(ResourceNotFoundException.class)
    public Result<Void> handleNotFound(ResourceNotFoundException e) {
        return Result.notFound(e.getMessage());
    }

    // 限流
    @ExceptionHandler(BlockException.class)
    public Result<Void> handleBlock(BlockException e) {
        return Result.tooManyRequests("请求过于频繁，请稍后再试");
    }

    // 兜底
    @ExceptionHandler(Exception.class)
    public Result<Void> handleUnknown(Exception e) {
        log.error("未预期的异常", e);
        return Result.error("服务器内部错误");
    }
}
```

### 3. 业务异常体系

```java
// 基类
public class BusinessException extends RuntimeException {
    private final int code;
    public BusinessException(int code, String t_message) {
        super(t_message);
        this.code = code;
    }
}

// 具体异常
public class ResourceNotFoundException extends BusinessException {
    public ResourceNotFoundException(String resource, String id) {
        super(404, resource + " 不存在: " + id);
    }
}

public class DuplicateResourceException extends BusinessException {
    public DuplicateResourceException(String resource, String key) {
        super(409, resource + " 已存在: " + key);
    }
}

public class TenantNotActiveException extends BusinessException {
    public TenantNotActiveException(String tenantId) {
        super(403, "租户已停用: " + tenantId);
    }
}
```
