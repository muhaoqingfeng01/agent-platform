# P5 交互端构建

> 对应 T1，负责 Web 聊天界面、审批卡片、用户反馈、IM 接入适配

## 子方案

| 序号 | 文件 | 内容 |
|:--:|------|------|
| 01 | [01-Web聊天界面.md](01-Web聊天界面.md) | React + WebSocket + 虚拟滚动 |
| 02 | [02-审批卡片.md](02-审批卡片.md) | 审批消息类型、卡片组件、交互流程 |
| 03 | [03-用户反馈.md](03-用户反馈.md) | 点赞/点踩、反馈数据收集 |

## 技术栈
- React 18+ (Vite)
- Ant Design 5.x
- WebSocket
- react-window
- 企业微信/钉钉 API

## 开发优先级
P5 — 前端交互层，可与后端 P0 并行开发，先做 Web → 再做 IM 适配
