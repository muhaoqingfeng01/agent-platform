# P5-04 IM 接入适配

## 概述
构建统一消息适配器层，将企业微信/钉钉等 IM 平台的消息格式转化为内部统一结构，实现跨渠道统一体验。

## 架构
```
企微 API ──┐
            ├── IM Adapter Layer ──→ 内部统一消息格式 ──→ 网关服务
钉钉 API ──┘
```

## 适配器接口
```java
public interface ImAdapter {
    /** 将 IM 平台消息转为内部统一格式 */
    UnifiedMessage convert(IncomingImMessage raw);

    /** 将内部回复转为 IM 平台格式并发送 */
    void sendReply(UnifiedMessage reply, String channelUserId);
}
```

## 支持渠道
| 渠道 | SDK/API | 消息类型 |
|------|---------|----------|
| 企业微信 | 企微回调 API | 文本/图片/卡片 |
| 钉钉 | 钉钉机器人 API | 文本/Markdown/卡片 |

## 回调对接
- 企微/钉钉回调 URL 统一指向网关 `/api/v1/im/callback/{channel}`
- 网关根据 `channel` 路由到对应适配器
- 鉴权方式：企微（corpId + token + aesKey）、钉钉（appKey + appSecret）
