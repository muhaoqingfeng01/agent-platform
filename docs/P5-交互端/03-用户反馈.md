# P5-03 用户反馈

## 概述
每条 AI 回复下方提供点赞/点踩入口，收集用户反馈用于效果评估和持续优化。

## 交互设计
- 每条助手消息气泡下方预设 👍 👎 图标
- 点击后即时切换选中态（再次点击取消）
- 点踩时弹出可选原因列表（不准确/不完整/不安全/其他）

## 消息格式
```json
{
  "type": "feedback",
  "messageId": "msg-uuid",
  "rating": 1,
  "reason": "不准确",
  "conversationId": "conv-uuid",
  "timestamp": "2026-06-13T12:00:00+08:00"
}
```
- `rating`: 1（点赞）/ -1（点踩）

## 后端处理
- 反馈数据写入 `t_evaluation_dataset` / `t_evaluation_dataset_item`
- 作为离线评估和 BadCase 闭环的数据源
- 与 P4-效果评估联动，触发优化工单
