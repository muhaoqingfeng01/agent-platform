## 关键决策

1. **过滤器链使用 Spring 自动注入** → `List<InputFilter>` 按 `order()` 排序
2. **Aho-Corasick** → Hutool WordTree 多模匹配，5 分钟刷新缓存
3. **审批推送** → 复用现有 ConversationWebSocketHandler
4. **DAG 集成** → ExecutionStatus 新增 WAITING_APPROVAL 状态，DagExecutionService 新增回调方法
5. **无新增 Flyway** → 3 张表已存在于 V1.1.0

## 踩坑记录

1. TenantContext 在 `infrastructure.context` 包，不在 `common.context`
2. SecurityFenceController 中不能用 `import ... as ...`（Java 不支持别名导入）
3. TaskExecution 只有 `@Getter`（无 `@Setter`），需添加行为方法而非 setter
4. Application 层直接引用 infrastructure 的 TenantContext/ConversationWebSocketHandler 在当前 DDD 实践中是允许的
