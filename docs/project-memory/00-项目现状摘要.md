## 十、已知限制和注意事项

1. **MySQL Connector 使用 8.0.33** — 不要升级到 9.x，Spring Boot 3.3.x 的默认兼容版本
2. **Spring AI Alibaba groupId 是 `com.alibaba.cloud.ai`** — 不是 `com.alibaba.cloud`
3. **Swagger 配置在 interfaces 模块** — `com.example.agent.interfaces.config.OpenApiConfig`
4. **Flyway 迁移按版本号顺序执行** — V1.0.0 → V1.1.0
5. **所有业务表含 tenant_id** — 多租户字段隔离
6. **Maven 使用 Aliyun 镜像** — 中建 Maven 仓库通过 `settings.xml` 配置
