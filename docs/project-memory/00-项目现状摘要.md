## 十、已知限制和注意事项

1. **MySQL Connector 使用 8.0.33** — 不要升级到 9.x
2. **Spring AI Alibaba groupId 是 `com.alibaba.cloud.ai`** — 不是 `com.alibaba.cloud`
3. **Swagger 配置在 interfaces 模块** — `OpenApiConfig.java`，路径 `/swagger-ui.html`
4. **SQL 迁移文件已移至 `docs/database/`** — 新环境部署时手动按版本顺序执行
5. **所有业务表含 tenant_id** — 多租户字段隔离
6. **Maven 使用 Aliyun 镜像** — 中建 Maven 仓库通过 `settings.xml` 配置
7. **日志配置由 logback-spring.xml 统一管理** — 不再使用 application.yml 的 `logging.*` 属性
8. **Session 快照在 `sessions/YYYY-MM-DD/` 目录** — 由 Stop Hook 自动生成
9. **P0 收尾已完成** — 4 个 RepositoryStub 已删除，ServiceImpl 已通过 @Primary 注入 RepositoryImpl，admin mock 已清除
