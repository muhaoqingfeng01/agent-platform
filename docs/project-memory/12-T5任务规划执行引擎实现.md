## 设计模式汇总（13 种）

| 模式 | 应用位置 |
|------|----------|
| **Strategy** | `ActionHandler` 接口 + 3 个实现 |
| **Chain of Responsibility** | `ActionHandlerRegistry` 委托到正确的 Handler |
| **Template Method** | `RetryPolicy.executeWithRetry()` 钩子 |
| **Mediator** | `DagExecutionService` 协调执行 + `ActionHandlerRegistry` |
| **Observer** | WebSocket 进度推送 |
| **Builder** | 所有实体/值对象（`@Builder`） |
| **Factory Method** | `IdGenerator.generate()` + `StepResult` 工厂方法 |
| **State** | `ExecutionStatus` / `StepStatus` 状态机 |
| **Memento** | `TaskStepExecution` input/output 快照 |
| **Command** | `TaskNode` 封装命令 + `ActionHandler` 执行 |
| **Facade** | `DagExecutionService` 简化 DAG 子系统 |
| **Composite** | `DagGraph` 组合多个 `TaskNode` |
| **Singleton** | `ActionHandlerRegistry` (Spring 单例) |
