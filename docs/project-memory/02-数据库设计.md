## 设计规范

1. **所有业务表**包含 `tenant_id` 字段（多租户隔离）
2. **统一字段**: `created_at`、`updated_at`、`deleted`（逻辑删除）
3. **引擎**: InnoDB，**字符集**: utf8mb4
4. **JSON 字段**用于灵活扩展（`input_json`、`output_json`、`metadata_json` 等）
5. **索引设计**覆盖常用查询维度
