## 关键决策

| 决策 | 理由 |
|------|------|
| McpClientManager 使用反射适配 Spring AI MCP API | 避免因 API 版本差异导致编译失败 |
| TenantContext 从 infrastructure.context 导入 | 项目所有 AppService 的既有约定 |
| 调用日志只追加不更新 | 审计合规，日志应为不可变记录 |
| ToolSchema/AuthConfig 在 Domain 层存为 POJO | DDD 原则：领域层技术无关 |
