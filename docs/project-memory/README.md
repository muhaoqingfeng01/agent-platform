## 会话记录目录结构

```
sessions/
├── .session-log.jsonl              ← 所有会话的追加日志（JSONL）
├── 2026-06-14/                     ← 按日期分目录
│   ├── session-2026-06-14T11-29-11+08-00.json
│   └── ...
└── 2026-06-15/
    └── ...
```

> 会话快照由 `.claude/hooks/session-snapshot.py` 在 Stop Hook 触发时自动生成，
> 仅在有文件变更时写入，无变更则静默跳过。
