# 会话记忆加载指南

## 下次会话启动时

在 Claude Code 或其他 AI 编码助手中，首先执行：

```
请先阅读 docs/project-memory/ 目录下的所有文件，了解项目背景和当前状态。
```

或直接指定：

```
请先阅读以下文件：
1. docs/project-memory/00-项目现状摘要.md
2. docs/project-memory/README.md
```

## 推荐加载顺序

1. **`00-项目现状摘要.md`** ← 最重要，包含所有核心信息
2. **`README.md`** ← 会话索引
3. 根据当前任务，选择性加载具体会话文件：
   - 遇到依赖问题 → `01-项目初始化与修复.md`
   - 数据库相关 → `02-数据库设计.md`
   - API 文档 → `03-Swagger集成.md`
   - 功能开发 → `04-开发计划与技术方案.md`

## 项目关键记忆点

1. **JDK 17 + Spring Boot 3.3.7** — 不要升级到 3.5.x
2. **Spring AI Alibaba 的 groupId 是 `com.alibaba.cloud.ai`** — 不是 `com.alibaba.cloud`
3. **MySQL Connector 用 8.0.33** — 不要用 3.0.33 或 9.x
4. **Swagger 配置在 interfaces 模块** — `OpenApiConfig.java`
5. **项目是纯脚手架** — 没有任何业务逻辑，所有 Controller/Service/Repository 待实现
6. **V1.1.0 迁移未执行** — Flyway 会在首次启动时自动执行
